## Installation
Pour l'execution de projet en local, install de python avec pip sont necessaire:

installation des dépendances :

```
cd MapboxCI2014
pip install -r requirements.txt
```

lancement du projet :

- *sur Linux*

```
python3 main.py
```
- *sur Windows*

```
python main.py
```
